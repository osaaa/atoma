## Color Palette

### Primary Colors:

- **Indigo/Purple:** Primary actions, branding
- **Green:** Completed habits, success states
- **Gray/Slate:** Default backgrounds, text
- **Red:** Delete actions, warnings

### Semantic Colors:

- **Completed:** Green (#10b981 or similar)
- **Incomplete:** Gray (#94a3b8 or similar)
- **Streak Fire:** Orange/Red gradient (#f97316 to #ef4444)
- **Progress Bar:** Gradient (indigo to purple)

---

## Typography

### Font Sizes:

- **Page Title:** 2xl (24px)
- **Section Headers:** xl (20px)
- **Habit Title:** lg (18px)
- **Body Text:** base (16px)
- **Small Text:** sm (14px)
- **Tiny Text:** xs (12px)

### Font Weights:

- **Bold:** Habit titles, section headers
- **Semibold:** Stats, numbers
- **Normal:** Body text, descriptions

---

## Spacing & Layout

### Card Spacing:

- **Padding:** 16px (p-4)
- **Gap between cards:** 16px (gap-4)
- **Border radius:** 12px (rounded-xl)

### Section Spacing:

- **Between sections:** 32px (my-8)
- **Within sections:** 16px (space-y-4)

### Responsive Grid:

- **Desktop:** 1 column (max-width: 800px centered)
- **Tablet:** 1 column (full width with padding)
- **Mobile:** 1 column (minimal padding)

---

## Interactions & Animations

### Hover States:

- Cards: Slight elevation (shadow increase)
- Buttons: Background color darkens
- Checkboxes: Border color changes

### Click Animations:

- Checkbox: Scale up slightly, then checkmark appears
- Card completion: Background color transition (0.3s ease)
- Streak update: Number counts up with animation

### Loading States:

- Skeleton cards while fetching data
- Spinner on action buttons during API calls
